== Description ==



This plugin has multiple slick images and carousel using shortcode. Its has so many features like responsive, swipe feature, infinite loop, mouse drag, auto play and many more. 

Its supported multiple sliders on any page. For each slider all slick slider options are customizable. We use jcolor to choose any color for Title, Title background, Description, Description background. 
Acquaint silk slider has facility to short slider images using drag drop facility.


==  Features == 

- Single slide

- Variable width slide

- Multiple

- Adaptive height

- Fade mode

- Responsive

- Center mode

- RTL

- Drag drop for sorting of slider images 

- Many more



== Installation ==

1. Upload `acq-slick-slider.zip` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Add new slider with required options in Acquaint Slick Slider

4. Add carousel in your content(Page, Post, Theme..)


You can add created sliders into any post,page or in theme using shortcodes.

E.g. [AQSS id=214] id = Slide ID.

You can find shortcode for each slider at listing page.

Multiple sliders at one page is possible, so it will be very useful for any user to see multiple sliders at one page.


== Frequently Asked Questions ==

= 1) How do I add  Acquaint Slick Slider to my theme? =

 The plugin works on shortcode and is independent of any theme. You will just need to put a short code on the single.php or you can directly add in content area of any page. Check installation steps for using shortcode of our plugin.


= 2) What happens to posts and pages when I delete a  Acquaint Owl Carousel slide? =

 It doesn't affect the page anyhow when you delete it. Just shortcode displayed as a text only, but it will not affect your other functionalities.

= 3) Can I use Post Acquaint Slick Slider with WordPress multisite? =

 Certainly, you can use it.
 
= 4) Can I arrange slider images? =

 Yes, the plugin has drag drop facility to sort slider images.

= 5) Where can I see all Acquaint Slick Slider slide shortcodes? =

 You can find it at Acquaint Slick Slider listing page.
